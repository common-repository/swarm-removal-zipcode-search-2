Swarm removal search by zipcode.

== Description ==

This plugin allows you to add a searchable database of beekeepers and companies that will remove swarms and established bee hives.  


Visit http://www.ohiostatebeekeepers.org/swarm-removal/ to see the plugin in use.

== Installation ==

1. Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation
2. Activate the Plugin from Plugins page

Insert the tag [swarmsearch] on the desired page/post.
